## 为什么不用 WSL2

官方 TRELLIS.2 目前只测试 Linux。本机首次启用 WSL2 的 `VirtualMachinePlatform` 后必须重启，但网吧还原系统会在重启后撤销安装，且网管无法关闭还原或添加白名单。因此使用第三方 Windows 适配版。

## 网吧还原后的手动恢复

需要 Python 3.11 64-bit 和 Git。在 PowerShell 逐条执行。

#### 1. ComfyUI 和虚拟环境

```powershell
New-Item -ItemType Directory -Path F:\yd-ai-3d -Force
Set-Location F:\yd-ai-3d
git clone https://github.com/Comfy-Org/ComfyUI.git
Set-Location F:\yd-ai-3d\ComfyUI
python -m venv .venv
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip setuptools wheel
```

#### 2. PyTorch 2.8 + CUDA 12.8

```powershell
python -m pip install torch==2.8.0 torchvision==0.23.0 torchaudio==2.8.0 --index-url https://download.pytorch.org/whl/cu128
python -c "import torch; print('Torch:', torch.__version__); print('Torch CUDA:', torch.version.cuda); print('CUDA:', torch.cuda.is_available()); print('GPU:', torch.cuda.get_device_name(0)); print('Capability:', torch.cuda.get_device_capability(0))"
python -c "import torch; a=torch.randn((4096,4096),device='cuda'); b=a@a; torch.cuda.synchronize(); print('GPU test OK:', b.shape, b.device)"
```

#### 3. ComfyUI 依赖和 TRELLIS.2 节点

```powershell
python -m pip install -r requirements.txt
git clone --recursive https://github.com/visualbruno/ComfyUI-Trellis2.git custom_nodes\ComfyUI-Trellis2
git -C custom_nodes\ComfyUI-Trellis2 submodule update --init --recursive
python -m pip install -r custom_nodes\ComfyUI-Trellis2\requirements.txt
```

#### 4. Torch 2.8 / Python 3.11 Windows wheel

```powershell
python -m pip install custom_nodes\ComfyUI-Trellis2\wheels\Windows\Torch280\cumesh-1.0-cp311-cp311-win_amd64.whl
python -m pip install custom_nodes\ComfyUI-Trellis2\wheels\Windows\Torch280\custom_rasterizer-0.1-cp311-cp311-win_amd64.whl
python -m pip install custom_nodes\ComfyUI-Trellis2\wheels\Windows\Torch280\flex_gemm-0.0.1-cp311-cp311-win_amd64.whl
python -m pip install custom_nodes\ComfyUI-Trellis2\wheels\Windows\Torch280\nvdiffrast-0.4.0-cp311-cp311-win_amd64.whl
python -m pip install custom_nodes\ComfyUI-Trellis2\wheels\Windows\Torch280\nvdiffrec_render-0.0.0-cp311-cp311-win_amd64.whl
python -m pip install custom_nodes\ComfyUI-Trellis2\wheels\Windows\Torch280\o_voxel-0.0.1-cp311-cp311-win_amd64.whl
```

验证：

```powershell
python -c "import torch, cumesh, flex_gemm, nvdiffrast.torch, nvdiffrec_render, o_voxel; print('Torch:', torch.__version__); print('All TRELLIS.2 extensions imported successfully')"
python -m pip check
```

#### 5. 验证网页

```powershell
$env:PYTORCH_CUDA_ALLOC_CONF="expandable_segments:True"
python main.py --listen 127.0.0.1 --port 8188
```

打开 <http://127.0.0.1:8188>，确认 `ComfyUI-Trellis2` 没有 `IMPORT FAILED`，按 `Ctrl+C` 停止。

## 下载模型

### 1. Hugging Face 登录和 F 盘缓存

```powershell
Set-Location F:\yd-ai-3d\ComfyUI
.\.venv\Scripts\Activate.ps1
New-Item -ItemType Directory -Path F:\yd-ai-3d\hf-cache -Force
$env:HF_HOME="F:\yd-ai-3d\hf-cache"
$env:HF_HUB_CACHE="F:\yd-ai-3d\hf-cache\hub"
python -m pip install --upgrade huggingface_hub
hf auth login
hf auth whoami
```

Token 只需 `Read` 权限：<https://huggingface.co/settings/tokens>

### 2. DINOv3（审批后）

```powershell
New-Item -ItemType Directory -Path models\facebook -Force
hf download facebook/dinov3-vitl16-pretrain-lvd1689m --local-dir models\facebook\dinov3-vitl16-pretrain-lvd1689m
Test-Path models\facebook\dinov3-vitl16-pretrain-lvd1689m\model.safetensors
Get-Item models\facebook\dinov3-vitl16-pretrain-lvd1689m\model.safetensors | Select-Object Name,Length
```

`Test-Path` 必须为 `True`。节点代码固定查找：

```text
F:\yd-ai-3d\ComfyUI\models\facebook\dinov3-vitl16-pretrain-lvd1689m\model.safetensors
```

### 3. TRELLIS.2-4B

```powershell
New-Item -ItemType Directory -Path models\microsoft -Force
hf download microsoft/TRELLIS.2-4B --local-dir models\microsoft\TRELLIS.2-4B
Test-Path models\microsoft\TRELLIS.2-4B\pipeline.json
Get-ChildItem models\microsoft\TRELLIS.2-4B -Recurse -File | Measure-Object Length -Sum
```
